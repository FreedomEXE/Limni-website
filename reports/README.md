# Reports

Generated outputs, research evidence, and verification artifacts.

## Contents

| Path | Purpose |
|---|---|
| `bias-gate/` | Bias-gate evidence that may be selectively deployed. |
| `data-verification/` | App versus indicator/export verification artifacts. |
| `archive/` | Older report outputs kept for traceability. |
| Other files/folders | Generated research, backtest, and audit outputs. |

## Rules

- Reports are evidence, not automatic runtime truth.
- Some app and script paths read specific report files; search before moving.
- New generated outputs should be reproducible, ignored, or tied to a named gate.
