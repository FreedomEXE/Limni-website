# Archive

Historical material lives here when it should stay available but should not sit
in an active project folder.

## Mirror Rule

Archive paths mirror repo ownership paths:

| Original area | Archive area |
|---|---|
| `docs/...` | `archive/docs/...` |
| `app/src/...` | `archive/app/src/...` |
| `app/scripts/...` | `archive/app/scripts/...` |
| `database/...` | `archive/database/...` |
| `config/...` | `archive/config/...` |
| `poseidon/...` | `archive/poseidon/...` |

Create deeper archive folders only when needed. Do not create per-folder archive
trees inside active areas.

## Rules

- Archived files are historical, not active truth.
- Repo evidence and current docs override archive material.
- Do not delete archived files without explicit approval.
- When moving files here, keep enough path context to show where they came from.
