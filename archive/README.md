# Archive

Root archive for inactive material.

Active folders should stay active. Stale handoffs, old planning notes, retired
assets, generated evidence that is no longer an active release artifact, and
legacy agent notes belong here.

## Contents

| Path | Purpose |
|---|---|
| `docs/` | Archived docs, handoffs, legacy agent notes, release planning notes, and old assets. |

## Rules

- Archive is not current truth.
- Do not cite archived material as active behavior without checking repo evidence.
- Preserve useful history here instead of leaving stale files in active folders.
